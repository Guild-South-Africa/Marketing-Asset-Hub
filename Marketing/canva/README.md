## What To Use

- Use `Marketing/png` for immediate posting on Instagram, LinkedIn, X, WhatsApp, and the website.
- Use `Marketing/svg` for print, signage, sponsor decks, and large-format production.
- Use `Marketing/canva/*.svg` when you want to import a design into Canva and edit the text, layout, or imagery.
- Use `Marketing/canva/canva-import-manifest.csv` to see the purpose, size, source, and export path for each asset.

## Canva Workflow

Import SVG files from this folder into Canva when you need editable designs. The campaign SVGs include embedded real GUILD SA source imagery and partner marks, so they should arrive with the actual visual assets present rather than empty placeholders.

## Design System

- 12-column modular grid
- 8px baseline rhythm
- Editorial spacing
- Flush-left hierarchy
- Oversized numerals as structural anchors
- Monochrome foundation with restrained GUILD SA accent color

## Export Folders

- SVG exports: `Marketing/svg`
- PNG exports: `Marketing/png`
- Canva editable sources: `Marketing/canva`
- Local source imagery used in the generated work is documented in `source-assets-contact-sheet.png`.
