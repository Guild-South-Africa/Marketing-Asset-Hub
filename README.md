# Marketing Asset Hub

GUILD SA AI Buildathon 01 — campaign asset generator, preview, and export tool.

| Path | Description |
|------|-------------|
| [`marketing-hub/`](marketing-hub/) | React app (see [README](marketing-hub/README.md)) |
| `Brand/` | GUILD SA logos and brand assets |
| `Event/` | Partner logos (Lovable, Eduvos, VelozTech) |
| `Stocks/` | Stock photography used in designs |
| `Marketing/canva/` | Template index and source SVGs |

## Quick start

```bash
cd marketing-hub
npm install
npm run dev
```
